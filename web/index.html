Usage

1. Select your desired subaccount to set as the root node for your visualisation.

2. Set the width of the visualisation. You can leave it blank and the default value will be used (2500), which fits most use-cases.

3. Zoom in and out to inspect the visualisation, and drag to pan across it.

4. Depending on your width value, some graphs (mostly smaller ones) may get out of focus. You're able to focus by zooming out and then back in.
